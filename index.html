Project Overview

This project presents an interactive Pakistan Super League Season 9 analytics dashboard built using Microsoft Power BI. It converts cricket match data into clear insights through KPI cards, charts, filters, and interactive visuals.

Dashboard Preview

Key Features

🏏 Batting Analysis

Top run scorers, strike rate, fours, sixes, batting average, and player performance comparison.

🎯 Bowling Analysis

Wickets, economy rate, bowling average, strike rate, wicket types, and bowler efficiency.

📊 Tournament Summary

Total runs, boundaries, matches, teams, venue distribution, team comparison, and match results.

Tools Used

Power BI Desktop · DAX · Power Query · Microsoft Excel · Data Modeling · Data Visualization
